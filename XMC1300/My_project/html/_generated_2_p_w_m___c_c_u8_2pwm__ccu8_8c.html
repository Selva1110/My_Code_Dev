PWM_CCU8_STATUS_t PWM_CCU8_SetDutyCycleAsymmetric ( PWM_CCU8_t *  handle_ptr,
uint32_t  shift,
uint32_t  duty_cycle 
)

Sets the shift and duty cycle in asymmetric mode.

Parameters
handle_ptrPointer to PWM_CCU8_t structure containing APP parameters.
shiftrequired shift
duty_cyclerequired duty
Returns
PWM_CCU8_STATUS_t
Description:
Sets the PWM shift and duty when APP is operating in asymmetric mode. The APP should not be in "PWM_CCU8_STATE_UNINITIALIZED" state. Shift and duty is scaled by 100.
The condition [shift + duty < 100%] should be met when APP operating in edge aligned mode.
The condition [shift < 50%] and [50% < shift + duty 100%] should be met when APP operating in center aligned mode.
Returns PWM_CCU8_STATUS_SUCCESS if operation is success.

Example Usage:

#include <DAVE.h>
int main(void)
{
PWM_CCU8_STATUS_t status;
DAVE_Init();
status = PWM_CCU8_SetDutyCycleAsymmetric(&PWM_CCU8_0, 3000, 4000);
while(1);
return 0;
}

Start the selected CCU8 slice.

Parameters
handle_ptrPointer to PWM_CCU8_t structure containing APP parameters.
Returns
PWM_CCU8_STATUS_t
Description:
Starts the selected CCU8 slice for PWM generation. Returns PWM_CCU8_STATUS_SUCCESS if the PWM_CCU8 APP state is in "PWM_CCU8_STATE_INITIALIZED" or "PWM_CCU8_STATE_STOPPED" else returns PWM_CCU8_STATUS_FAILURE.
PWM_CCU8_Start() is needed to be called if "Start during initialization" is unchecked to start PWM generation, else its called by DAVE_Init();

{
DAVE_Init();
//This API needs to be called if "Start during initialization" is unchecked
PWM_CCU8_Start(&PWM_CCU8_0);
while(1);
return 0;
}
