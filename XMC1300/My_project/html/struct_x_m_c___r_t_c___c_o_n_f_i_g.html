Detailed Description

RTC initialization with time, alarm and clock divider(prescaler) configurations

The structure presents a convenient way to set/obtain the time and alarm configurations
for RTC. The XMC_RTC_Init() can be used to populate the structure with the time and alarm values of RTC.
